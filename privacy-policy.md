# Privacy Policy – Quick Tweet

Effective date: February 2026

Quick Tweet is a Chrome extension designed with a single purpose:  
to allow users to compose and post tweets to X using a distraction-free popup.

## Data Collection

Quick Tweet does **not** collect, store, transmit, or sell any personal data.

- No analytics are used
- No tracking is performed
- No user data is sent to any external server

## Tweet Content

- Tweets are posted directly through the official X (x.com) web interface
- Tweet content is **not stored, logged, or accessed** by the developer
- All posting actions occur locally in the user’s browser

## Local Storage

Quick Tweet may store minimal user preferences locally in the browser, such as:
- Theme selection (dark or light mode)

This data:
- Remains on the user’s device
- Is never transmitted externally
- Can be removed by uninstalling the extension

## Third-Party Websites

The extension interacts only with:
- `https://x.com`
- `https://twitter.com` (for legacy redirects)

These interactions are required solely to enable tweet composition and posting via the official X web interface.

## User Control

All actions performed by Quick Tweet are:
- Explicitly initiated by the user
- Limited to composing and posting a tweet

The extension does not run in the background without user interaction.
